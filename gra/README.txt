SellFix: Telefon w Potrzebie

Jak uruchomić:
1. Rozpakuj paczkę ZIP.
2. Otwórz plik index.html w przeglądarce.
3. Kliknij Start gry.

Sterowanie:
- Komputer: W/S albo strzałki góra/dół.
- Telefon: przyciski ▲ ▼ pod grą.

Pliki:
- index.html - główny plik gry
- style.css - wygląd gry
- game.js - logika gry

To jest pierwsza gotowa wersja demo. Można ją dalej rozbudować o mapę Kielc, logo SellFix, sklep, system zleceń, minigry naprawy i tryb tycoon.
